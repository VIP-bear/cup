# 校园二手平台
## 项目描述
构建一个二手交易平台，用户可以发布二手商品，管理员在后端审核通过后，可以购买商品。
该系统由二手交易和后台管理审核两部分组成。后台管理主要负责对于用户、发布商品的审核管理功能；
二手交易系统主要提供商品的浏览和搜索，并提供用户下单购买。
## 技术栈
### 前端
-   Vue
-   ElementUI
-   HTML、CSS、JavaScript
### 后端
-   Spring、Spring MVC、Spring Boot
-   MySQL、JPA
## 功能
-   登录/注册
-   发布商品
-   审核商品
-   删除商品
-   搜索商品
-   加入购物车
-   购买商品
-   用户管理
